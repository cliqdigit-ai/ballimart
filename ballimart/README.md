# 🛒 BallMart — Ballia ka Apna Grocery App

Zepto-style grocery delivery app for Ballia, UP.

## Features
- 🏠 Home page with rotating banners, categories, popular products
- 🔍 Search page — sabhi products dhundho
- 🛒 Cart with coupon (BALLIA50), address form, payment options
- 📦 Order success with live tracking animation
- 📋 Order history page
- 👤 Profile page

## FREE Deployment on Vercel (Step by Step)

### Step 1 — GitHub pe upload karo
1. [github.com](https://github.com) par account banao (free)
2. "New Repository" click karo — naam do: `ballimart`
3. Is poore folder ko upload karo (drag & drop works)

### Step 2 — Vercel pe deploy karo
1. [vercel.com](https://vercel.com) par jao — "Sign up with GitHub"
2. "Add New Project" → apna `ballimart` repo select karo
3. Bas "Deploy" click karo — 2 minute mein live ho jayega!
4. Free URL milegi: `ballimart.vercel.app`

### Local mein chalane ke liye
```bash
npm install
npm start
```
Browser mein khulega: http://localhost:3000

## Coupon Code
